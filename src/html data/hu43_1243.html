    Start Button Finally Returns: Wndows 8.1

    In advance of next month's Build conference, Microsoft today provided a "first look" at Windows 8.1, which includes the return of the start button.
    With Windows 8.1, the first major update to Microsoft's Windows 8 operating system, the start button will appear whenever you move your mouse to the bottom right of the screen in tiled (or "Metro") mode and will persist on the taskbar when in desktop mode.
    To make it more easily recognizable, Microsoft will replace the Start "tip" that appears on Windows 8 with the familiar Windows logo.

    The anti-Windows 8 circle jerk is one of my favorites. I've heard about people who buy a new decked out PC but only get Windows 7 with it. The only thing is when asked they can't explain why they hate Windows 8 aside from the old regurgitated messages like "No start button..." or "It's made for a tablet, not a desktop!"

    Windows 8 has:

    *Better security.
    *Increased functionality in a variety of programs and games.
    *Higher levels of compatibility with software.
    *Significantly shorter start up and shut down times.
    *Better multimedia performance.
    *Higher browser performance.
    *Far, far, far better multi-language support.

    Windows 7 has:

    *A start button.

    The vast majority of people cannot think for themselves and therefore will just repeat whatever opinion they heard on the matter first. What's so hard about actually looking into something by yourself and coming to your own conclusions? It literally takes under 30 seconds to install an interface on Windows 8 and you have your precious start button back. There are several options to choose from, including some that are free and others that cost $5 or less.

    Windows 7 was a very well built operating system, and Windows 8 essentially built on it with a new approach. If you don't like it, you can just download an interface and have something identical to Windows 7 with better functionality. I don't say it's worth the money to upgrade to Windows 8 if you're not all that interested in it, but it's not a horrible OS to be avoided at all costs.
